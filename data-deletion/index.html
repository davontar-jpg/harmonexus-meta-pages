User Data Deletion Instructions

Harmonexus HMIE uses authorized platform APIs for Harmonexus media operations, account connection checks, publishing metadata, and approved output workflows.

Users can request deletion of data associated with the Harmonexus HMIE app by emailing harmonexus11@gmail.com with the subject line Harmonexus HMIE Data Deletion Request.

Please include the platform name, account handle or page name if applicable, and a brief description of the data you want reviewed for deletion.

Requests will be reviewed and processed manually. HMIE does not ingest media from Instagram, Facebook, or TikTok. YouTube remains the only ingestion source.
